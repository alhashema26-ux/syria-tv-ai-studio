# نيوميديا — إلهامي (Inspire)

**النوع:** محتوى رقمي يبني هوية القناة ويلهم الجمهور
**الجمهور:** متابعون مخلصون للقناة
**النبرة:** ملهمة، ثقافية، تبني علاقة طويلة الأمد
**عمر المحتوى:** أشهر وسنوات

## تعليمات توليد العناوين
- ركز على القيمة المضافة والمعنى العميق
- الأسلوب: Curiosity Gap أو Pattern Interrupt
- اربط المحتوى بهوية سوريا وتراثها

## تعليمات السوشيال ميديا
- فيسبوك: قيمة ثقافية عميقة
- إنستغرام وتيك توك: بصري ملهم
- كل المنصات بأسلوب مختلف
